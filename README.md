# Astory Portfolio Scroll Page

这是一个用于作品集网站的 Astory 长屏滚动案例页。

## 文件结构

```text
astory.html
css/astory.css
js/astory.js
gameasset/
```

## 使用方式

1. 把整个文件夹放进你的作品集项目。
2. 用浏览器打开 `astory.html`。
3. 把 `gameasset` 里的占位 SVG 替换成你的真实截图或设计图。
4. 如果你的返回地址不是 `index.html?page=2`，修改 `astory.html` 顶部的 `.back-link`。

## 图片替换建议

保留同名替换最省事：

```text
gameasset/astory_cover.svg
gameasset/screen_home.svg
gameasset/screen_market.svg
gameasset/screen_people.svg
gameasset/screen_rebirth.svg
gameasset/system_map.svg
gameasset/android_structure.svg
```

可以把 SVG 换成 PNG/JPG，但要同步修改 HTML 里的文件名。
