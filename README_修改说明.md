# 修改说明

这份包包含 2–7 六套风格页面的修改版：

- 2 怪盗：第三页改为「怪盗陈列室」，新增图片在 `2image/`。
- 3 游戏：第三页改为「项目选关」，新增图片在 `3image/`。
- 4 野蛮：第三页改为「项目任务看板」，新增图片在 `4image/`。
- 5 展馆：第三页改为「展厅平面导览」，新增图片在 `5image/`。
- 6 瑞士：第三页改为「网格索引」，新增图片在 `6image/`。
- 7 杂志：第三页改为「编辑目录」，新增图片在 `7image/`。

同时修改了 JS：第三页不再拦截滚轮/A/D 做划船，滚轮与方向键恢复正常翻页；新的项目卡片会沿用原本的项目进入转场。

使用方式：把这些文件和文件夹复制到你的网站根目录，覆盖原来的 `2index.html`、`2style.css`、`2script.js` ... `7script.js`，并保留你原项目中的 `assets/`、项目详情页和 `loading-transition.js` 等原有文件。
