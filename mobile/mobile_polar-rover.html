00 / OVERVIEW

PolarArk
极地车全流程设计

以“低变化高纬度地区”为场景，从背景调研、用户痛点、外饰造型、内饰体验、健康座舱系统到 HMI 原型，构建可长期停留的移动生活系统。

01 / BACKGROUND RESEARCH

问题从“低变化环境”开始。

高纬度地区的极夜、弱日照、低纹理景观与单调声环境，会让人长期处于低唤醒状态。设计需要补偿感知、情绪和生活节律。

02 / USER JOURNEY

出行本身，正在变成消耗。

从出发前准备、途中驾驶、停留生活到返程复盘，用户痛点不只是车辆性能，而是漫长环境下的疲惫、孤独、信息不确定与生活断裂。

03 / DESIGN STRATEGY

把交通工具，变成可运行的生活系统。

策略围绕移动、居住、环境补偿、社交/独处切换展开，让车辆成为高纬度地区可长期停留的生活单元。

04 / EXTERIOR CONCEPT

外饰从虎鲸与冰海中提取力量感。

形态语言参考虎鲸、浮冰与极地海面，建立厚重、防护、灵动之间的视觉关系。

05 / EXTERIOR OUTPUT

从形态迭代到车辆规格。

车辆外观强调包覆感、通过性和生活舱体量，最终输出外饰渲染与规格说明。

06 / INTERIOR EXPERIENCE

内饰从 Survival 转向 Living。

内饰不是简单“能活下去”，而是把休息、社交、独处、恢复和感官补偿组织成可持续的生活体验。

07 / HEALTH CABIN SYSTEM

森林循环呼吸，是内饰体验的系统核心。

健康座舱系统通过空气、光照、声景与情绪反馈，抵消长期极地环境带来的低唤醒和疲惫感。

08 / HMI ARCHITECTURE

HMI 是驾驶员的“第三只眼”。

HMI 架构把环境信息、车辆状态、风险提示和生活舱控制集中组织，降低高风险环境中的认知负担。

09 / HMI PROTOTYPE

从系统架构，落到可交互界面。

界面围绕风险、路线、能源、健康座舱和任务状态展开，手机端可进入轻量 HMI 演示。
